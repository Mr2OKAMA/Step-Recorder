# Step Recorder (Tango Light)

Webブラウザ上の操作（クリックやテキスト入力）を追従し、自動的に美しい手順書（HTML/PDF）を作成するブラウザ拡張機能（Edge / Chrome対応）です。

## ✨ 主な機能
* **マウス追従ハイライト**: クリック対象の要素を自動で赤枠ハイライト。
* **自動スクショ＆ドキュメント化**: クリックした瞬間の画面キャプチャと、操作内容のテキストを自動記録。
* **テキスト入力の記録**: フォームに入力した内容もタイムラインに自動挿入。
* **シンプル操作パネル**: 画面左下のミニマルなコントローラー（✓ ⏸ 🗑）で直感的に操作。
* **美しく印刷可能なレポート**: 完了時にワンクリックでPDF保存・印刷可能なレポートを生成。

## 🚀 インストール方法（無料配布用）

1. 本リポジトリの **[Code]** ボタン（緑色）をクリックし、**[Download ZIP]** を選択してダウンロードします。
2. ダウンロードした ZIP ファイルをPC上の好きな場所に解凍します。
3. ブラウザ（Edge または Chrome）を開き、拡張機能の管理画面を開きます。
   * **Edge**: アドレスバーに `edge://extensions/` と入力
   * **Chrome**: アドレスバーに `chrome://extensions/` と入力
4. 画面内にある **「デベロッパー モード」** のスイッチを **ON** にします。
5. **「展開して読み込む」**（または「パッケージ化されていない拡張機能を読み込む」）ボタンをクリックします。
6. 先ほど解凍したフォルダ（`manifest.json` が入っているフォルダ）を選択すると、インストールが完了します。

## 🛠️ 使い方
1. 手順を作成したいWebサイトを開き、一度ページを再読み込み（F5）します。
2. ブラウザのツールバーにある **Step Recorder** のアイコンをクリックし、**「✨ キャプチャを開始」** を押します。
3. 画面左下に操作パネルが出現し、マウスを動かすと赤枠が追従します。
4. 通常通り操作（クリック・入力）を行います。
5. 終わったら左下の **`✓` (完了)** ボタンを押すと、自動的に手順書レポートが別タブで開きます。
   * `⏸` ボタンで一時停止、`🗑` ボタンで記録を破棄できます。
